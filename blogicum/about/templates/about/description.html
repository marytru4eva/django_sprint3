О чём этот проект

Проект «Анфиса для друзей» представляет собой каталог мороженого, созданный для изучения веб-разработки на Django.

Что можно делать

  • Просматривать список мороженого
  • Управлять категориями, топпингами и обёртками через админку
  • Добавлять новые сорта мороженого

Технологии

  • Django 3.2
  • SQLite
  • Bootstrap для стилей
